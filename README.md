## Getting Started

To run this project locally:

1. Clone the repo:
    ```bash
    git clone https://github.com/WDT1203/portfolio.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portfolio
    ```
3. Open `index.html` in your browser.

> No build steps required; this is a static site.
